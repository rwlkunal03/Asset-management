# Smart Asset Management Platform

A full-stack web application for managing cultural council assets at IIT Roorkee.

## Features
- Login system with Admin and User roles
- Browse available assets (cameras, mics, costumes etc.)
- Book assets with date picker
- Admin can approve or reject bookings
- Analytics dashboard with charts

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: Supabase
- Charts: Recharts

## How to Run

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm run dev
